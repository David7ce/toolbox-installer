# Toolbox Installer

Astro-based website to explore list of packages and generate install commands across different categories:

- Desktop OS
- Mobile OS
- Browser extensions
- VS Code extensions
- Software Libraries

## Requirements

- Node.js 18+
- npm
- Astro with TS

## Installation

```sh
# 1. Install dependencies:
npm install

# 2. Start development server:
npm run dev

# 3. Build for production:
npm run build

# 4. Preview production build:
npm run preview
```

## Main Structure

- src/pages: Site pages
- src/components: Reusable components
- src/scripts: Client-side interaction logic
- public: Static assets
- pkgs: Package lists in JSON format
